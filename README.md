# STM32-ADC-Read-using-Register-Level-Programming-Bare-Metal-
Bare-metal STM32F4 project demonstrating ADC1 configuration and analog signal acquisition from PB0 (ADC1_IN8) using register-level programming. Includes GPIO analog mode setup, ADC channel selection, software-triggered conversion, and polling-based data acquisition (12-bit resolution) without using HAL/LL libraries.
